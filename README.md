# 学生成绩分析与可视化项目

## 项目简介
本项目是一次**学生学业成绩数据的完整分析与可视化实践**。
通过对学生成绩表进行数据清洗、统计分析、等级划分和图表展示，完成了一套标准化的数据分析流程。

## 项目功能
  数据加载与预处理（清洗无效数据、缺失值处理）
  学生总分、平均分计算与排名
  各科最高分、最低分统计
  成绩等级自动分箱（不及格、及格、中等、良好、优秀）
  多科目成绩分布可视化图表
  标准化工程化项目结构

## 技术工具
  Python
  Pandas（数据处理）
  Matplotlib（数据可视化）
  VS Code（开发工具）
  GitHub（代码托管）

## 数据来源
  原始数据：`1.2025-2026学年第一学期学业成绩表.xls`
  数据格式：Excel 表格
  数据内容：某班级学生多门课程的学业成绩信息

## 数据说明
  原始数据已做匿名化处理，仅包含课程成绩信息，不涉及任何可识别的学生个人隐私信息。
  本项目仅用于教学与学习实践，严禁用于其他用途。
